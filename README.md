# 📦 Inventory Management System

A full-stack web application designed to manage inventory efficiently, track product transactions, and analyze stock data in real time.

---

## 🚀 Project Overview

The **Inventory Management System** is built to simplify stock management for businesses. It allows users to manage products, monitor inventory levels, and track sales/purchase transactions through an intuitive interface.

---

## 🛠️ Tech Stack

### 🔹 Frontend

* React.js
* HTML5, CSS3, JavaScript

### 🔹 Backend

* Spring Boot (Java)
* RESTful APIs

### 🔹 Tools & Technologies

* Maven
* Git & GitHub
* VS Code & Eclipse IDE

## ⚙️ Installation & Setup

### 🔹 Backend Setup (Spring Boot)

1. Open project in Eclipse
2. Update Maven dependencies
3. Run the application
4. Backend runs on:

```
http://localhost:9191
```

---

### 🔹 Frontend Setup (React)

1. Open terminal in `inventory-front`
2. Install dependencies:

```
npm install
```

3. Run the app:

```
npm start
```

4. Frontend runs on:

```
http://localhost:3131
```
